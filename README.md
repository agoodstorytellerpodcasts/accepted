# OmniReach

An all-in-one marketing powerhouse that delivers real, engaged followers and likes across Instagram, TikTok, YouTube, X (Twitter), and Facebook — plus search-engine visibility on Google and Firefox.

## Project Structure

This repository contains the following components of the OmniReach platform:

- `frontend/`: The React/Vite dashboard providing the user interface for campaign management, analytics, and AI chat.
- `backend/`: The Fastify API server handling business logic, user authentication, and data persistence with SQLite.
- `website/`: The Express-based marketing website for OmniReach.
- `ai-chat/`: The AI Chat WebSocket service powered by OpenAI for real-time marketing strategy and performance analysis.
- `architecture.md`: Comprehensive platform architecture blueprint.
